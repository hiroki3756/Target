## サイト概要
自分が達成したい目標を立て、達成度の記入や一言コメントを追加できるサイト

### サイトテーマ
目標を投稿してそれに対して達成度の記入、コメントの追加ができる、そのほかにも他のユーザーのフォロー機能、フォローしたユーザーの目標を閲覧できる機能、
お問い合わせ機能を搭載したサイトです

### テーマを選んだ理由
目標というものは、なかなか達成できないものなので、達成を支援できるサイトがあればいいなと思ったからです

### ターゲットユーザ
全年齢

### 主な利用シーン
達成したい目標がある人
目標を立てたが、なかなか達成できない人
